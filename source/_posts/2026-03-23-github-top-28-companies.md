## 新增大模型公司详情

### Anthropic
- **GitHub地址**：https://github.com/anthropics
- **核心项目**：
  - Claude Code：终端AI编程助手,理解代码库并执行自动化任务（78.9K stars）
  - Skills：Agent技能公共仓库（95.3K stars）
  - Claude Cookbooks：Claude使用示例和最佳实践集合（35.1K stars）
  - Claude Plugins：官方插件目录（12.4K stars）
  - 多语言SDK：Python、TypeScript、Go、Java、C#、Ruby等
  - Prompt Engineering Tutorial：交互式提示工程教程（33.5K stars）
- **技术特色**：
  - Claude系列模型：Claude 3.5 Sonnet、Claude 3 Opus等
  - AI安全优先：专注于 Constitutional AI 和安全对齐
  - 开发生态完善：从SDK、教程到插件生态
  - Agent能力突出：Skills框架和Code工具赋能智能体开发

### Google DeepMind
- **GitHub地址**：https://github.com/google-deepmind
- **核心项目**：
  - AlphaFold 3：蛋白质结构预测,生物AI里程碑项目
  - AlphaFold 2：开源版蛋白质折叠预测工具（27K+ stars）
  - MuJoCo：多关节动力学物理模拟器（16K+ stars）
  - Acme：强化学习组件和代理库（10K+ stars）
  - Optax：JAX梯度处理和优化库（7K+ stars）
  - RecurrentGemma：基于Griffin的开源权重语言模型
- **技术特色**：
  - 326个仓库,覆盖强化学习、科学计算、优化等多个领域
  - JAX生态建设：dm-haiku、distrax等JAX生态核心库
  - 学术界影响力：DeepMind Research仓库配合论文发布
  - 跨学科突破：从AlphaGo到AlphaFold,再到核聚变模拟

### Mistral AI
- **GitHub地址**：https://github.com/mistralai
- **核心项目**：
  - Mistral-Inference：官方Mistral模型推理库
  - Mistral-Finetune：模型微调工具
  - Mistral-Common：Mistral模型预处理库
  - Cookbook：使用示例和实践指南
  - 多语言客户端：Python、TypeScript客户端库
  - Megablocks：稀疏MoE模型训练框架（与Databricks合作）
- **技术特色**：
  - 法国AI公司,欧洲开源AI代表
  - 高效模型设计：Mistral 7B、Mixtral 8x7B等参数高效的模型
  - 开源策略积极：开源权重和推理工具
  - 企业级支持：SageMaker等云平台集成

### xAI
- **GitHub地址**：https://github.com/xai-org
- **核心项目**：
  - Grok-1：Grok开源版本（51.5K stars）
  - X-Algorithm：X平台推荐算法,Rust实现（16.1K stars）
  - xAI SDK：官方Python API SDK（386 stars）
  - xAI Cookbook：Grok API实战示例（357 stars）
  - Grok Prompts：Grok聊天助手和X bot的提示词库（4K stars）
  - xAI-Proto：gRPC API的公共protobuf定义
- **技术特色**：
  - Elon Musk创立,强调理解宇宙
  - Grok系列：Grok-1、Grok-2等大语言模型
  - X平台深度集成：推荐算法开源、平台生态整合
  - 实时信息获取：利用X平台实时数据增强模型能力

### Moonshot AI（月之暗面）
- **GitHub地址**：https://github.com/MoonshotAI
- **核心项目**：
  - Kimi-K2.5：最强大的开源多模态Agent模型
  - Kimi-K2：1T总参数的MoE模型,32B激活参数（10.5K stars）
  - Kimi-Linear：混合线性注意力架构（1.3K stars）
  - Kimi CLI：下一代CLI智能代理（7.2K stars）
  - Attention-Residuals：Transformer残差连接改进（1.9K stars）
  - Kosong：现代AI代理应用的LLM抽象层
- **技术特色**：
  - 专注于长上下文和Agent能力
  - Mooncake：KV-centric disaggregated LLM服务（FAST 2025最佳论文）
  - Kimi-Dev：开源编程大模型,SWE-bench Verified达60.4%
  - 多模态布局：Kimi-VL、Kimi-Audio等

### MiniMax AI
- **GitHub地址**：https://github.com/MiniMax-AI
- **核心项目**：
  - MiniMax-01：456B总参数,459B激活参数,支持400万token上下文
  - MiniMax-M1：全球首个开源权重大规模混合注意力推理模型
  - MiniMax-M2.5：编程专用模型,SWE-Bench Verified达80.2%
  - Mini-Agent：专业级单Agent演示框架
  - MiniMax-MCP：官方MCP服务器,支持TTS、图像/视频生成
- **技术特色**：
  - Lightning Attention机制：线性复杂度处理长文本
  - 108天三连更(M2→M2.1→M2.5),迭代速度业界罕见
  - 成本仅为Opus/GPT-5的1/10到1/20
  - 多模态能力：文本、视觉、语音全覆盖

### Zhipu AI（智谱AI）
- **GitHub地址**：https://github.com/THUDM（清华大学KEG实验室）
- **核心项目**：
  - GLM-5：从Vibe Coding到Agentic Engineering
  - GLM-4.5：Agentic, Reasoning, Coding基础模型
  - ChatGLM系列：ChatGLM3、ChatGLM2-6B（40.7K stars）
  - CogVLM/CogAgent：多模态视觉语言模型和GUI Agent
  - CodeGeeX：代码生成模型
  - Open-AutoGLM：开源手机端智能助理框架
- **技术特色**：
  - 源自清华大学技术成果,中文效果优异
  - GLM-4.5总参数355B,激活32B,参数效率极高
  - 开源生态活跃,文档丰富,社区支持强大
  - Zread工具：GitHub项目一键生成高质量使用手册
